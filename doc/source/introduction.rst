Data Share==========
Access to the tomographic data share is implemented via `Globus <https://www.globus.org/>`_.
To access the data enter `request <https://github.com/lorentz-phantom/lorentz-phantom/issues/>`_.

All data sets can be access and reconstructed using `tomoPy <http://tomopy.readthedocs.org/>`_ 
:cite:`DeCarlo:14b`, :cite:`Gursoy:14a` and `ASTRA <https://sourceforge.net/projects/astra-toolbox/>`_ 
:cite:`vanAarle201535`, :cite:Palenstijn2011250`.
